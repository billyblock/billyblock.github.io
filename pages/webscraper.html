NBA Average Salary per Country.

Web Scraper

This project scrapes NBA player nationalities and salary from multiple websites, combining the datasets, and displays the average player salary by country.

View on GitHub

Project Overview

This project uses Python and Selenium to navigate webpages, extract the data, and store it in JSON. Afterwards, it displays the info using Matplotlib.

Features:

  • Scrapes player names and nationalities from RealGM.
  • Scrapes player salaries from HoopsHype.
  • Matches names across both data sources.
  • Stores the collected data in JSON files.
  • Calculates the average salary by country.
  • Generates a bar chart of the top earning countries.
